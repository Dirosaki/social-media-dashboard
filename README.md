## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Dirosaki/social-media-dashboard
```

Entre no diretório do projeto

```bash
  cd social-media-dashboard
```

Instale as dependências

```bash
  yarn install
```

Inicie o servidor

```bash
  yarn dev
```

## Stack utilizada

**Front-end:** React, Vite, TypeScript e Styled-Components
